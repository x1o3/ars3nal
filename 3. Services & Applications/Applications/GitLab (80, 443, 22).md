## GitLab Footprinting & Enumeration

- The only way to footprint the GitLab `version number` in use is by browsing to the `/help` page when logged in.
- Some GitLab istances may `allow user registration` without confirmation from an administrator
- **Authenticated**: browsing to `/explore` we can check for any `public projects` that may contain something interesting

---

## GitLab User Enumeration

We can write a script ourselves in Bash or Python or use [this one](https://www.exploit-db.com/exploits/49821) to enumerate a list of valid users. The Python3 version of this same tool can be found [here](https://github.com/dpgg101/GitLabUserEnum).

**Gitlab < 16.6**:
Login attempt are limited to 10 before a waiting of 10 minutes.

**Gitlab >= 16.6**:
It is the same BUT we can modify this in the admin UI.
